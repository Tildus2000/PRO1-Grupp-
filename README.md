# Från sensor till Graf 

Detta repo handlar om hur man bygger en DHt 11 sensor, bygger en server samt samlar in datan så den kommer upp i realtid på en graf.

## Material: 
För att utföra detta behöver du:
* Ett elektronik och lödningskit som innehåller följande:
-  En microcontroller
-  Ett breadbord
-  En 5000 resistor
-  DHT 11 sesnor
-  sladdar
-  En USB/ USB-C kabel kompatibel med din dator + microcontorller

* Ett kodnings
